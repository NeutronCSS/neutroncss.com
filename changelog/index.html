Changelog

1.0

Flexbox Support

Released on 18 December 2015

Added support for Flexbox

Added equivalent mixins to support creating layouts with flexbox.

  • flex-columns(): Uses almost identical parameters as floatbox's columns().
  • flex-order(): Identical syntax to floatbox but uses the native flexbox order property.

Ability to override default configuration

You can now change the default body width without needing to edit Neutron's files directly. Simply set any of the below variables before including Neutron in your project.

Check the documentation for details.

// General Settings
$neutron-column-padding: 4px 8px;
$neutron-container-max-width: 960px;
$neutron-flush-margin: true;
$neutron-flush-padding: false;

// Breakpoint Settings
$neutron-mobile-max: 479px;
$neutron-phablet-max: 767px;
$neutron-tablet-max: 1023px;
$neutron-desktop-sml-max: 1199px;
$neutron-desktop-mid-max: 1799px;

Breakpoint improvements

Custom breakpoints can now be created using any valid CSS media query criteria. Syntax of custom breakpoints has changed, see documentation for details.

Breaking Changes

  • container-align() is now align-container() to match similar flexbox syntax.
  • The $container-align property for columns() is now $align-container
  • Syntax for custom breakpoints has changed. Check the documentation for details.
